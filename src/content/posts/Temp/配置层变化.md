# 2026-03-14

**配置层变化**
你最该先看的是这些配置文件，因为这里最容易出现“上游加了新字段，你本地没配”或者“旧字段已经被替换”的情况。

- siteConfig.ts
  新增了 pageWidth、card.followTheme、navbar.menuAlign、bangumi.categoryOrder、pages.gallery、categoryBar、imageOptimization.noReferrerDomains。
- siteConfig.ts
  postListLayout.grid.columns 已经被 postListLayout.grid.columnWidth 取代，如果你还按老的 columns 配，就属于过时写法。
- navBarConfig.ts
  导航结构变了，上游加了“我的”分组，并把 Gallery 和 Bangumi 放进子菜单里。
- friendsConfig.ts
  旧的 columns 配置被移除，改成了 title、description、showCustomContent、randomizeSort。
- sidebarConfig.ts
  position 现在支持 "right"，新增了 tabletSidebar，旧的 showRightSidebarOnPostPage 被 showBothSidebarsOnPostPage 取代。
- backgroundWallpaper.ts
  全屏透明模式新增了 overlay.switchable、overlay.cardOpacity，blur 逻辑也增强了。
- commentConfig.ts
  waline 新增了 emoji 配置。
- expressiveCodeConfig.ts
  新增了 pluginLanguageBadge.enable。
- galleryConfig.ts
  这是新文件，上游新增了相册系统。
- sponsorConfig.ts
  message 字段从类型里去掉了，不建议再继续配。

**类型定义变化**
如果你后面继续自己改配置，最好也顺手看类型文件，能直接知道哪些字段是上游当前支持的。

- config.ts
  新增了 GalleryConfig、GalleryAlbum、LinkPreset.Gallery。
- config.ts
  FriendsPageConfig、SidebarLayoutConfig、SiteConfig、CommentConfig、BackgroundWallpaperConfig 都有扩展。
- config.ts
  明确移除了或替换了部分旧结构，比如友链页列数配置、网格列数字段、赞助留言字段。

**页面和内容结构变化**
这部分和你站点内容定制关系很大，尤其是友链页。

- friends.mdx
  上游把旧的 friends.md 改成了 MDX 版本，结构更复杂，但可定制性更强。
- friends.astro
  现在会读取 friendsPageConfig.title、description、showCustomContent，并用自适应网格代替旧列数控制。
- about.md
  上游只加了一条 QQ 联系方式，结构变化不大。

**上游新增的主要功能**
这些不是单纯“修修补补”，而是你可能会想主动决定开不开、配不配的新功能。

- 相册系统
  相关文件：galleryConfig.ts、index.astro、[album].astro
- 分类快捷导航栏
  相关文件：siteConfig.ts、CategoryBar.astro
- 评论区快速跳转按钮
  相关文件：BackToComment.astro
- 推荐文章组件
  相关文件：RecommendedPost.astro
- 加密文章 / 密码保护
  相关文件：EncryptedPost.astro、crypto-utils.ts
- 图片防盗链处理
  相关文件：image-utils.ts、siteConfig.ts
- 透明模式增强
  相关文件：DisplaySettingsIntegrated.svelte、backgroundWallpaper.ts
- 音乐管理增强
  相关文件：MusicManager.astro、musicConfig.ts

**你最值得手动复查的文件**
如果你准备自己再细调一遍，我建议按这个顺序检查：

1. siteConfig.ts
2. navBarConfig.ts
3. sidebarConfig.ts
4. friendsConfig.ts
5. backgroundWallpaper.ts
6. commentConfig.ts
7. galleryConfig.ts
8. friends.mdx
